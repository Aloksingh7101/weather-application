# 🌤️ Weather Hub Application

A responsive weather web application built using **HTML, CSS, and JavaScript**. The application allows users to search for any city and instantly view real-time weather information, including temperature, weather conditions, humidity, and wind speed through a weather API.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* ☁️ Current weather conditions
* 💧 Humidity information
* 🌬️ Wind speed details
* 📍 Weather data based on user input
* 📱 Responsive design for desktop and mobile

---

## ⚙️ How It Works

1. Enter the name of any city.
2. Click the **Search** button.
3. The application sends a request to the Weather API.
4. The API returns real-time weather data.
5. Weather details are displayed instantly on the screen.
